# 🌟 Tablas Mágicas

App web para que niños de 8 años aprendan las tablas de multiplicar jugando.

## ✨ Características

- **📖 Modo aprender**: cada multiplicación se muestra con dibujos (grupos de emojis) para entender visualmente qué significa multiplicar.
- **🎮 Modo juego**: quiz de 10 preguntas con 3 vidas, respuestas con 4 opciones y apoyo visual.
- **🏆 Modo reto**: preguntas mezcladas de todas las tablas (del 1 al 10).
- **⭐ Progreso con estrellas**: cada tabla guarda las estrellas conseguidas (se guardan en el dispositivo).
- **🔊 Voz**: botón para escuchar la multiplicación en español.
- **📱 Optimizada para tablet**: botones grandes, diseño táctil y responsive.

## 🚀 Uso

Es una app 100% estática, sin dependencias. Abre `index.html` en el navegador o sírvela con cualquier servidor estático.

## 🛠️ Tecnologías

HTML + CSS + JavaScript vanilla. Sin frameworks, sin build.
