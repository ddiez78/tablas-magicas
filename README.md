# 🚀 Tablas Mágicas — ¡La aventura de multiplicar!

App web gamificada para que niños de 8 años aprendan las tablas de multiplicar jugando.

**En vivo:** https://tablas-magicas-six.vercel.app

## 🎮 Juegos

- **👾 Batalla de jefes**: derrota a 10 monstruos distintos respondiendo multiplicaciones antes de que se acabe el tiempo. Respuestas rápidas = golpe crítico ⚡.
- **⚡ Rayo 60s**: contrarreloj de 1 minuto con combos multiplicadores (x2…x5) y cohete de progreso. Guarda tu récord.
- **🧠 Parejas**: juego de memoria uniendo cada operación con su resultado.
- **📖 Aprender**: cada tabla con dibujos (grupos de emojis) y voz en español.

## 🏆 Gamificación

- **🪙 Monedas** por cada acierto, con efectos flotantes y sonidos.
- **✨ XP y niveles** con barra de progreso y subidas de nivel celebradas.
- **🛍️ Tienda de mascotas**: 10 mascotas coleccionables (🦉→🦁) que se compran con monedas; la elegida te acompaña en el menú.
- **🔥 Racha diaria** de días jugados.
- **⭐ Estrellas** por tabla según lo bien que juegues.
- Combos, confeti, banners y sonidos WebAudio (sin archivos externos).

Todo el progreso se guarda en el dispositivo (localStorage).

## 🛠️ Tecnologías

HTML + CSS + JavaScript vanilla en un solo archivo. Sin frameworks, sin build. Desplegada en Vercel.
